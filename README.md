# GoodDollar Daily Faucet
## 🚀 Overview
This project is a dedicated faucet-style application designed to drive user engagement and token distribution within the **GoodDollar ecosystem** on the **Celo blockchain**. Built as a Monorepo, this tool serves as a utility-focused engine to incentivize consistent activity and participation, mirroring successful engagement patterns seen in SocialFi protocols.
## 📊 Gamified Reward Structure
We have implemented an **Infinite Milestone Loop** system to reward consistent users. Every daily check-in earns a **+10 G$ base bonus**, with additional milestone rewards that scale as you maintain your streak.

## 🛠 Key Features
 * **Celo Blockchain Integration:** Engineered specifically for the Celo network to leverage its fast and efficient transaction capabilities.
 * **Faucet Distribution System:** Automated, faucet-style token distribution designed to provide ongoing value to community members.
 * **User Engagement Engine:** Built to foster consistent daily interaction, helping to bootstrap activity for token-based projects.
 * **Modular Architecture:** Structured as a Monorepo for maintainability and scalability, including a mock sandbox environment for rapid testing and development.
 * **Community-First Design:** Developed with the goal of creating real utility for GoodDollar tokens, drawing on best practices from current SocialFi and blockchain engagement models.
